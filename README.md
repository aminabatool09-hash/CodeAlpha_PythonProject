# CodeAlpha Python Projects

This repository contains the Python tasks I completed during my internship at CodeAlpha. The purpose of these projects is to practice basic programming concepts and improve problem-solving skills.

## Tasks Included

1. Hangman Game  
A simple word guessing game where the user tries to guess a hidden word letter by letter within limited attempts.

2. Stock Portfolio Tracker  
A basic program that allows the user to enter stock names and quantities, and then calculates the total investment based on predefined prices.

3. Basic Chatbot  
A simple chatbot that responds to user inputs such as greetings and basic questions using conditional statements.

## Technologies Used

- Python
- Basic programming concepts such as loops, conditions, and user input

## How to Run

1. Install Python on your system  
2. Open terminal or command prompt  
3. Run any file using the following command:

python filename.py
